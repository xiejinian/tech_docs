02 · 合规边界（股票内容专项）

分享认知，不踩红线

股票投资内容是最大的合规敏感区。我们的核心定位是「高胜率投资者的认知分享者」，而非「投资顾问」。以下是可执行的合规框架。

⚖️ 核心定位原则：「我分享我的认知，你做你的决策」
所有股票相关内容，始终以「个人真实投资经历的主观认知分享」为框架输出。强调「这是我个人的思考过程」而非「你应该这样做」。这一定位在法律上与「证券投资咨询」有本质区别，是合规的知识付费内容形式。
🔄
表达方式转换对照表
同样的内容，换一种表达方式，合规性截然不同
❌ 不能这样说（荐股语气）→✅ 可以这样说（认知分享语气）
「XX 股现在可以买，目标价 50 元」 → 「我自己在关注 XX 行业，我个人判断这类公司在当前阶段有一定的估值修复空间，但我不做任何预测」
「现在是抄底好时机，快买」 → 「根据我过去的经验，当 A 指标和 B 指标同时出现时，我个人会加仓，但每个人的风险承受能力不同」
「我判断下周会涨，建议持有」 → 「我分享一下我目前的持仓框架和背后的逻辑，至于市场会怎么走，没有人能确定」
「跟我操作，三个月回报 30%」 → 「我复盘一下我今年的投资决策过程，哪些判断对了、哪些错了，以及错在哪里」
「这支股稳赚，明天就买」 → 「我个人对这个行业长期看好的逻辑是 XXX，但短期价格走势不可预测，这不是建议」
「放心跟买，我的胜率有 70%」 → 「我分享一下我的选股框架和仓位管理方法，至于你是否适合，需要根据自己情况判断」
✅
投资内容可以做的
📖
分享个人真实持仓框架（不具体到买入成本和仓位比例）
🧠
输出对某行业、某板块的个人宏观判断和逻辑推演
📉
复盘自己的历史交易决策（包括失败的），分析原因
🔍
分享选股的思考框架、估值方法、仓位管理原则
🌐
分析宏观政策、产业趋势、市场情绪对股市的影响
💬
在私密群中进行非公开的个人观点实时交流
📊
介绍技术分析工具和方法（不做预测，只讲工具本身）
📚
推荐投资书籍、研究报告（不推荐具体标的）
❌
绝对不能做的
🚫
以任何形式公开推荐具体股票代码的买入或卖出时机
🚫
承诺或暗示跟随操作的收益率，如「跟我买稳赚」
🚫
在公开平台（抖音/小红书/公众号/星球）发布具体荐股信息
🚫
收取基于绩效的费用（如跟买成功后抽成利润）
🚫
代客理财或管理他人资金进行投资
🚫
传播未经证实的内幕消息或小道消息
🚫
私密群内发布「明天开盘买 XX」等具有明确买卖指令性质的内容
🚫
对电报群成员的投资损益承担任何形式的责任保证
📌 私密群（微信/电报）的特殊合规处理
1. 入群协议（必须签署）：成员入群前须确认"本群内容仅为个人经验交流，不构成任何投资建议，本人自行承担投资决策责任。"

2. 群公告置顶：长期置顶免责声明，所有内容为个人真实经历分享，非投资顾问服务。

3. 发言口径规范：投资专家在群内发言始终以「我个人的想法是…」「我自己会这样考虑…」「仅供参考，不构成建议」为标准口径。

4. 截图传播管控：群规明确禁止截图外传，电报群可开启防截图设置，降低内容被断章取义传播的风险。
