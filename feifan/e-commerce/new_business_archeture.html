业务域层
📦 商品域 Product
商品管理 商品展示 商品详情 类目管理 SKU管理
商品底层服务 Feign
商品 CRUD · 价格计算 · 类目树 · 品牌管理 · SPU/SKU 模型
🗄 MySQL (商品库) + ES (商品搜索)
🛒 交易域 Trade
购物车 正向交易 订单管理 逆向管理 物流跟踪
交易底层服务 Feign
下单流程 · 订单状态机 · 退换货 · 物流对接 · 结算拆单
🗄 MySQL (订单库) + Redis (购物车)
📊 库存域NEW Inventory
库存管理 库存扣减 预占释放 仓库管理
库存底层服务 Feign
实时库存 · 预占 / 扣减 / 回滚 · 多仓路由 · 超卖防护 (Redis Lua)
🗄 MySQL (库存库) + Redis (热点库存)
👤 用户域NEW User
会员管理 账户中心 地址管理 积分体系
用户底层服务 Feign
注册/登录 · 会员等级 · 收货地址 · 积分流水 · OAuth2
🗄 MySQL (用户库) + Redis (Session)
🎯 营销域 Marketing
营销管理 优惠计算 活动管理 优惠券
营销底层服务 Feign
满减 · 折扣 · 优惠券核销 · 活动商品池 · 规则引擎
🗄 MySQL (营销库) + Redis (券/活动缓存)
💳 支付域 Payment
支付管理 C端支付 退款管理 对账管理
支付底层服务 Feign
支付宝/微信/银行卡对接 · 退款 · 对账 · 资金流水
🗄 MySQL (支付库，独立实例)
核心跨域调用链路 （Feign RPC · Sentinel 熔断保护）
交易 → 商品 查询商品信息 / 价格
交易 → 库存 预占 / 扣减库存
交易 → 用户 获取收货地址 / 会员信息
交易 → 营销 优惠计算 / 券核销
交易 → 支付 创建支付单
支付 → 交易 支付回调 / 更新订单状态
营销 → 商品 活动商品池关联
