01 · 战略全景

三个方向，一个系统

三个商业方向之间存在弱协同关系——渠道为网关提供成本优势，社群为网关提供高质量客源，但各方向均可独立造血，无强依赖。

方向 01
🌐
AI 企业网关
面向国内出海企业和 To B/To C 用户的通用 AI 接入网关，GitHub 是渠道之一，不依赖单一上游。提供统一的 API 接入、用量管理、计费服务。
核心产品 To B / To C 持续造血 香港主体
方向 02
📈
投资创业社群
以连续创业者 + HR/猎头专家背景为核心，面向中文用户和海外华人的投资与创业知识社群。首选知识星球或私域社群模式，冷启动阶段从零开始。
知识变现 华人华人 独立造血 弱协同

💰

产品定位与差异化
为什么选择我们而不是直接对接官方
价格优势
通过折扣渠道谈判，提供低于官方定价的 API 使用成本，最低保持 20% 空间传递给客户。
合规与稳定性
香港主体 + 海外 API 接入，解决国内企业使用海外大模型的支付、合规、访问稳定性难题。
多厂商聚合
统一接入 Claude、GPT-4o、Gemini 等，单一 API Key 切换多模型，无需重复对接。
中文本地化支持
中文文档、中文客服、人民币结算（通过香港主体转换），大幅降低国内企业使用门槛。
企业级功能
团队用量管理、子账号隔离、用量告警、账单明细、权限控制，满足企业采购与合规要求。

03 · 方向二

投资创业知识社群

以连续创业者 + 资深 HR & 猎头专家的组合背书，打造面向华人群体的高质量投资与创业知识付费社群。

🎯
社群定位
差异化：实战 + 资源对接 + 人才
核心差异化：三位一体
投资视角（财务配置）+ 创业实战（连续创业经验）+ 人才资源（猎头能力），三者结合在同类社群中极为稀缺。
目标人群
有一定积累、寻求财务配置优化的中高端华人用户；正在创业或计划创业的技术从业者；海外华人群体（有汇率配置需求）。
内容支柱
① 投资配置方法论（基础 → 进阶）② 创业实操复盘 ③ 人才招募 & 团队建设 ④ 出海 + 跨境资源对接。
❄️
冷启动方案
从零开始，3 个月内跑通付费模型
第 1 月
蓄水期
私域积累 + 内容试水
创始人微信朋友圈、公众号开始持续输出（每周 2-3 篇）
从现有人脉邀请 50 人进免费测试群，收集内容需求
知识星球开通，免费邀请 100 人，设定星球人设和规则
确定 3 个核心内容选题，做第一批精华内容
第 2 月
付费转化
首批付费用户转化
推出首批限时优惠价（早鸟价），目标首批 50 付费用户
安排首次线上直播分享，主题：连续创业者的财务配置复盘
建立内容更新节奏：每周深度帖 × 2 + 问答 × 1
设计邀请裂变机制（老带新）
第 3 月
规模化
扩圈 + 变现多元化
付费用户目标 200+，尝试调整到正常价
探索高客单价服务：一对一咨询、小班课
引入 AI 工具推荐（与网关方向联动，但非强依赖）
评估是否拓展海外华人群体（小红书、Twitter/X）

04 · 方向三

AI 折扣渠道体系

同步推进官方渠道 + 第三方渠道谈判，打通 Google、OpenAI、Anthropic Claude 等头部厂商的商业折扣，构建整体成本护城河。

🏛️
官方渠道策略
正在推进，预期 30-60 天落地
Anthropic Claude 官方渠道
申请成为授权渠道合作伙伴，获取批量用量折扣。香港主体有助于绕开地区限制，优先推进。
Google Cloud / Vertex AI
通过 Google Cloud 合作伙伴计划（Reseller 或 MSP 路径），获取 Gemini 系列模型的批量折扣及返利。
OpenAI 批量协议
申请 OpenAI 的企业用量折扣计划，承诺最低月消费量换取折扣。评估是否通过 Azure OpenAI 路径补充。
GitHub 企业订阅通道
探索作为 GitHub Copilot 企业版的渠道分销商，为开发团队提供更便捷的团队订阅购买路径。
注意事项
GitHub Copilot 反向工程不可行（已深度调研）：所有开源逆向工程项目均声明不可商用，官方更新频繁（每月 50+ 次），滥用检测主动封号。唯一合法路径是正式渠道合作。
🔗
第三方渠道策略
并行推进，补充官方覆盖不到的场景
现有合作方资源（红熊 AI）
已有国内阿里云 + 海外 API 算力资源（Claude/GPT/Gemini），含 Token 中继平台源码及商业使用权，可作为第三方渠道的核心底层。
多级分销网络
向具有更强议价能力的上游批发商采购，作为中间层分销给终端企业客户，形成合理的价差收益。
渠道合规材料管理
海外 API 中继的合规材料需提前备齐（参考红熊 30 天内提供）。确保转售行为符合各平台服务条款，降低封号风险。
风险分散原则
不依赖单一渠道，官方 + 第三方并行；多厂商覆盖（非单一 OpenAI 依赖），任意一个渠道出问题时可快速切换。
📋

主要风险与应对预案

渠道谈判失败
高
官方折扣谈判周期超预期，或条件不满足最低毛利要求
应对：优先落地第三方渠道作为保底，同步推进多家厂商并行谈判
香港主体合规
中
香港公司注册 + 银行开户周期较长（银行 KYC 严格），影响收款
应对：提前 2 个月启动，准备充分的 KYC 材料；备选虚拟银行（众安等）
上游服务封号
高
转售行为触发官方 TOS 违规，导致 API Key 被封
应对：仅走正规渠道合作，备齐合规材料；多厂商分散，单一封号不影响整体
社群冷启动慢
中
投资专家朋友没有现成受众，内容吸引力验证需要时间
应对：先做私域积累，再开付费；内容聚焦实战复盘，差异化定位
大客户周期长
低
企业采购决策链长，网易同事的大客户拓展周期可能超过 3 个月
应对：大客户收入不计入前 3 月财务目标，以中小开发者为现金流基础
三人协同效率
低
三人分布在不同地方，网易同事兼职参与，协同成本高
应对：每周固定同步会（1小时），明确 RACI，关键决策文档化

个人收益隔离原则
参照萌招科技的经验教训：多重身份（个人合伙人 vs 商业主体）的收益必须严格隔离，分开纳税。

香港公司开户建议

传统银行（推荐）
汇丰 / 恒生 / 渣打。开户周期 4-8 周，材料要求高但信用背书强，适合对接 Stripe、Google Cloud 等大厂付款。
虚拟银行（快速启动备选）
众安银行 / WairBank / Neat。开户 1-2 周，KYC 宽松，适合快速启动收款，后续补充传统账户。
支付通道
Stripe（海外收款）+ 支付宝 / 微信国际版（国内客户人民币充值），通过香港主体合规换汇。
